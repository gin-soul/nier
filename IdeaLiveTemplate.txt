
File ---> Settings ---> Editor ---> LiveTemplates
点击add,先添加分组TemplateGroup;再在分组中点击add,添加LiveTemplate

if-else是最常见的分支语句
提示符：ife
模板代码：
if($END$){}else{}

提示符：eif
模板代码：
else if($END$){ 

}

提示符：els
模板代码：
else{ 
	$END$
}


方法模板:
可以分别定义四个:pum,pusm,prm,prsm
提示符：pum
模板代码：
public $TYPE$ $NAME$() {
	$END$
}
提示符：pusm
模板代码：
public static $TYPE$ $NAME$() {
	$END$
}
提示符：prm
模板代码：
private $TYPE$ $NAME$() {
	$END$
}
提示符：prsm
模板代码：
private static $TYPE$ $NAME$() {
	$END$
}


字段模板:
提示符：pf
模板代码：
/**
* $COMMENT$
*/
@Getter
@Setter
private $TYPE$ $NAME$;

idea自带模板:pfsi pfss :private final static int 与 private final static String 的简写
